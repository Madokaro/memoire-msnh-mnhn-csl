# memoire-msnh-mnhn-csl
Style CSL pour les mémoires du Master MSNH (MNHN), adapté pour Zotero.
Ce dépôt contient un style CSL destiné aux mémoires du Master MSNH du MNHN.
Il a été conçu pour Zotero et respecte les exigences de citation et de mise en forme demandées pour le mémoire de M2 du master.

Fonctionnalités principales :
- Format auteur-date adapté
- Gestion spécifique des :
- vidéos en ligne (YouTube, Vimeo, conférences filmées)
- films
- lois, décrets, décisions de justice
- entrées de dictionnaire et encyclopédie
- pages web
- Mise en forme conforme aux normes du mémoire MSNH
- Compatibilité complète avec Zotero
  
Installation :
- Téléchargez le fichier memoire-msnh-mnhn.csl
- Ouvrez Zotero → Edition → Préférences/Paramètres → Citer
- Cliquez sur + (à droite de Obtenir d'autres styles) puis sélectionnez le fichier .csl que vous venez d'enregistrer
- Le style apparaît dans la liste sous le nom Mémoire MSNH MNHN
